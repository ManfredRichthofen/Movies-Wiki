# Jellyfin

jfapp hosts a community Jellyfin server with thousands of movies and shows. **Seerr** is connected to the server — search for a title and request it if it is missing, then stream on your phone, TV, or browser. No subscription required.

**Server address:** `jfapp.xyz`
