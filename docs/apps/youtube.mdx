# YouTube for Android

Ad-free YouTube with background playback, SponsorBlock, and other premium-style features — installed as an APK on your Android device.

**New here?** Start with the [installation guide](/docs/installation) — it covers downloads, MicroG, and sign-in step by step.

## What you get

- No video ads
- Background playback
- Picture-in-picture
- SponsorBlock (skips sponsored segments)
- Custom themes

## Extra troubleshooting

**Video plays but no sound**

- In the app settings, turn off **Direct play** or choose a lower stream quality.

**App won’t update**

- Download the latest **YouTube** APK from the [Apps page](/Apps) and install it over the old version. If that fails, uninstall the old app first, then install the new APK.

## Updates

Check the [Apps page](/Apps) periodically for new builds. When YouTube updates on Google’s side, older APKs may stop working until a new build is published.
